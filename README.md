# OpenTelemetry Learning Resources

> Educational repository for learning OpenTelemetry — Cloud-Native Observability & Instrumentation

Part of **CNCF Project Focus** series - exploring cloud-native technologies in depth.

## About This Repository

This repository contains learning materials and educational examples for **OpenTelemetry**, the open-source observability framework graduated by the CNCF.

**Educational focus.** Real code and examples.

### [carousel/](./carousel)

Collection of visual presentations (carousels) illustrating OpenTelemetry concepts.

### [opentelemetry-first-sample/](./opentelemetry-first-sample)

Complete hands-on tutorial to get started with OpenTelemetry on Kubernetes, including:

- OTel Collector deployment with Helm (kube-opentelemetry-stack)
- Collector pipeline configuration (receivers, processors, exporters)
- Zero-code auto-instrumentation for Python & Java applications
- Distributed tracing across a 3-service e-commerce demo
- Head and tail sampling strategies for cost optimization
- Kyverno policies for observability governance
